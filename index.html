Projects

Chess AI

"Shallow Blue" — Github

A complex chess bot that uses minimax and alpha-beta pruning to decide on the best move. Written from scratch in python, this project took about a month to get working and another month to optimize. The program can beat almost any casual player using the default settings. The UI features a captured-piece visual, as well as a coloured box around the previous move played.

Tetris-like Game

"Quadris" — Github

A tetris-inspired game with the aim of clearing as many rows as possible. Built using C++ by me and two other friends, the game features a hint and an auto-saved high score. There are also 4 levels of increasing difficulty. It varies from classic Tetris because of the command-based gameplay, which allows the user to queue up several moves at once.

Vote-based Social Network

"StreetRep" — Github

A social network that allows users to post status updates at specific locations. Based on yik-yak, the site gives users a certain amount of cred based on upvotes and downvotes. This project was built at Hack the North 2016 using node.js, firebase, and bootstrap.

Automated Rap Assistant

"Rapatronics" — Github

An Android app that suggests rhyming lines based on voice input from the user. The backend uses Google speech-to-text technology, as well as various natural language processing API's. This app was a contender for QHacks 2017 and was made with the help of three other people.
